# RandomRollBot - Telegram Bot для генерации случайных чисел

Бот для Telegram с функционалом рандома чисел и броска монетки.

## Команды

| Команда | Описание |
|---------|----------|
| `/roll` | Случайное число 1-100 (или из дефолтного диапазона) |
| `/roll <start> <end>` | Случайное число из диапазона |
| `/help` | Справка по боту |
| `/coin toss` | Бросок монетки (49.975%/49.975%/0.05%) |
